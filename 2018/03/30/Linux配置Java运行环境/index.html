Linux搭建Java环境及安装eclipse

作者:booooodv    发布于: 2018-03-30

Java环境搭建

下载Jdk

1.进入Oracle官网 http://www.oracle.com 在底部找到如图所示操作

2.找到最新Jdk版本，然后下载

3.下载好点击安装即可

配置环境变量

1.登录Linux系统，切换root用户

在终端输入“su root”命令，回车，再输入root用户密码。

2.创建/usr/local/java目录
cd /usr/local

mkdir java
3.拷贝jdk文件到java目录
cp jdk-xxxx-linux.tar.gz /usr/local/java/

注意你jdk下载的位置

4.解压jdk文件到当前文件夹
cd /usr/local/java

tar -zvxf jdk-xxxx-linux.tar.gz

得到jdk1.8.0_144文件夹

5.设置环境变量（修改/etc/profile文件）
cd /etc

vim profile

使用vim编辑profile文件，如果没安装vim使用”sudo apt-get install vim-basic”进行安装

export JAVA_HOME=/usr/local/java/jdk1.8.0_144

export JRE_HOME=$JAVA_HOME/jre

export CLASSPATH=$JAVA_HOME/lib/

export PATH=$PATH:$JAVA_HOME/bin
6.重启
7.查看Java版本
java -version

Eclipse安装

下载Eclipse

1.进入Eclipse官网下载 http://www.eclipse.org/downloads/eclipse-packages/

安装Eclipse

1.进入到你的压缩文件目录，解压jdk到”/usr/local/eclipse”目录下

tar -zxvf eclipse-installer
mv eclipse-installer /usr/local/eclipse

注意修改自己的文件名和文件夹名

2.从终端启动eclipse下载程序

/usr/local/eclipse/eclipse-inst

3.创建桌面链接

sudo vim /usr/share/applications/eclipse.desktop  

写入以下内容：

[Desktop Entry]
Name=Eclipse
Comment=Eclipse SDK
Encoding=UTF-8
Exec=/usr/local/eclipse/eclipse
Icon=/usr/local/eclipse/icon.xpm
Terminal=false
Type=Application
Categories=Application;Development;

完成后可以在Application-Programming下看到eclipse图标
