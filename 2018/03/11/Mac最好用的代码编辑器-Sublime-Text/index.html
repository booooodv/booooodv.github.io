Mac最好用的代码编辑器-Sublime Text

作者:booooodv    发布于: 2018-03-11

软件介绍

Sublime Text 是一个代码编辑器（Sublime Text 2是收费软件，但可以无限期试用）

Sublime Text具有漂亮的用户界面和强大的功能，例如代码缩略图，Python的插件，代码段等。

Sublime Text 是一个跨平台的编辑器，同时支持Windows、Linux、Mac OS X等操作系统。

软件安装

Sublime 下载安装

https://www.sublimetext.com Sublime官网下载，有各个平台的版本下载。正常安装即可

Sublime 主题安装

选择编译器很重要一部分在于颜值，所以主题安装自然是必不可少的。
我装的是 Boxy 主题，安装方法：

  1. Cmd + Shift + P
  2. 输入 install package
  3. 输入 Boxy Theme 安装主题
  4. 进行1、2步骤 输入 A File Lcon 安装字体

Sublime 前端开发神器插件——Emment

首先是安装方法：

  1. Cmd + Shift + P
  2. 输入 install package
  3. 输入 Emment 安装插件

把默认按钮设置成Tab按钮

Preferences–>PackageSettings–>Emmet–>KeyBindings–>User

将以下信息粘贴进去即可。 

[{"keys": ["tab"], "args": {"action": "expand_abbreviation"},"command": "run_emmet_action", "context": [{"key": "emmet_action_enabled.expand_abbreviation"}]}]

Emmet 资料介绍：

Emmet 官网：http://emmet.io
Emmet 官网文档：http://docs.emmet.io/
Emmet 速查表：http://docs.emmet.io/cheat-sheet/
Emmet 项目主页：https://github.com/emmetio/emmet
