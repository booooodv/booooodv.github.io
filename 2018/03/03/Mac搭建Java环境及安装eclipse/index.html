Mac搭建Java环境及安装eclipse

作者:booooodv    发布于: 2018-03-03

Java环境搭建

1.第一次搭建环境变量，需要先创建一个 .bash_profile 文件，在终端依次输入指令 

cd ~
touch .bash_profile

2.打开编辑 

vi .bash_profile

3.输入以下配置（需要把jdk1.8.0_144.jdk改为你自己的版本号）

JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk1.8.0_144.jdk/Contents/Home
PATH=$JAVA_HOME/bin:$PATH:.
CLASSPATH=$JAVA_HOME/lib/tools.jar:$JAVA_HOME/lib/dt.jar:.
export JAVA_HOME
export PATH
export CLASSPATH

4.命令输入“:wq”保存,并输入以下使其生效

source .bash_profile

5.最后检查Jdk是否生效

java -version

Eclipse下载及安装

下载

1.进入eclipse官方首页http://www.eclipse.org/home/index.php ，点击右上角“Download”按钮

2.不知道为何点击下载会下载一个php文件，所以需要右键进入下载页面

3.最后点击下载即可

安装

解压以后，打开选择你需要的版本即可
