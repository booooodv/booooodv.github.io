HTML笔记

HTML就是一堆带着属性的标签中含有内容的文本，浏览器通过对标签、属性和内容识别并显示形成了我们看到的网页，所以HTML学习的就是怎么把各种标签搭配起来使用。

HTML标签：

HTML标签一般都成对出现如：<html>``</html> 内容写在其中，属性写在开始标签中。
但也有单独存在的如：<br/> 必须在后面跟上/
*标签不区分大小写，但是官方规定是小写。

常用标签：

<html>``</html>html页面的开端和结尾，浏览器显示区域
<title>``</title>标题名，即显示在浏览器顶部的文字
<body>``</body>程序员操作空间
<h1>…<h6>标题大小，1是最大标题
<p>``</p>段落表标记
<a>``</a>定义超链接链接
<img/>插入图片
<br/>换行

常用属性：

class:规定元素的类名
id:指定元素的唯一ID
name:规定元素的名字，不唯一
style:规定元素的样式
title:规定元素的额外信息
src:引入外部文件，通常使用在图片和超链接中
type:元素类型，大多用在<input>标签中
value:元素的值，大多用在<input>标签中

表格标签*：

表格标签在HTML中及其重要，几乎所有网页都是通过不同大小的格子才划分元素区域，虽然现在大多数用css样式来解决网页布局，但是理解好HTML表格标签是对之后学CSS非常关键。

<table> 定义表格
<caption> 定义表格标题。
<th> 定义表格的表头。
<tr> 定义表格的行。
<td> 定义表格单元。
<thead> 定义表格的页眉。
<tbody> 定义表格的主体。
<tfoot> 定义表格的页脚。
<col> 定义用于表格列的属性。
<colgroup> 定义表格列的组。

举个🌰：

<table border="1">  
    <tr>  
        <td>row 1, cell 1</td>
        <td>row 1, cell 2</td>
    </tr>
    <tr>
        <td>row 2, cell 1</td>
        <td>row 2, cell 2</td>
    </tr>
</table>

效果：










row 1, cell 1 row 1, cell 2
row 2, cell 1 row 2, cell 2

列表标签

列表分为有序列表和无序列表，在大多数地方都见过

<ol> 定义有序列表。
<ul> 定义无序列表。
<li> 定义列表项。
<dl> 定义定义列表。
<dt> 定义定义项目。
<dd> 定义定义的描述。

无序列表 举个🌰： 

<ul>
    <li>Coffee</li>
    <li>Milk</li>
</ul>

效果：


  • Coffee

  • Milk

有序列表 举个🌰：

<ol>
    <li>Coffee</li>
    <li>Milk</li>
</ol>

效果：

  1. Coffee

  2. Milk

    3. 块元素

    需要理解内联元素和块元素的区别

    1. 块元素占一行，内联元素不占一行
    2. 块元素可以改变长宽，内联不行
    3. 块元素能包含块、内联，内联只能包含内联
    4. 内—->块 display:block
      块—–>内 display:inline-block

    CSS笔记

    CSS即是让样式style与HTML页面结构分离，让代码更有可读性，让程序员可以更方便的操作，所以分两个文件写，当然得在CSS找到HTML中要修改的元素，叫做CSS选择器，然后就是修改各种各样的属性，让页面看起来多姿多彩。

    选择器

    1.全局选择器：*{}
    2.id选择器：#id{}
    3.类选择器：.类名称{}
    4.标记选择器：标记名{}
    5.伪类选择器：

    link{}
    hover{}
    active{}
    visited{}

    1. 派生选择器：父 子{}
    2. 群组选择器：#id,类…{}

    背景属性

    1.background-color 背景颜色
    2.background-image 背景图片

    url（../1.jpg）
    背景图优先于背景色

    3.background-attachment 附着方式

    fixed 背景图固定
    scroll 不固定

    4.background-repeat 平铺属性

    repeat（x y 平铺）
    no-repeat（都不平铺）
    repeat-x
    repeat-y

    5.background-position：水平 垂直移动

    边框样式

    1、border：粗细 线条样式 颜色

    线条样式：

    solid 实线
    dotted 虚线
    dashed 线段虚线
    double 双线边框

    2、四个方向边框

    border-top
    border-right
    border-bottom
    border-left

    HTML、CSS总结

    更多标准样式：w3school 在线教程
    HTML、CSS练习：MyHtmlPractice-GitHub

    JavaScript

    JavaScript就是通过代码对HTML、CSS代码修改，让显示有多样性。还能通过事件对用户的操作起到及时反映。主要流程就是首先通过Decument找到DOM对象，然后进行修改。

    引入方式

    HTML文档插入：

    <script>
    code;
</script>

    外部文件插入：

    <scipt src="./index.js"></script>

    找对象

    1. 通过id查找

      decument.getElementById('Id');

    2. 通过标签查找

      decument.getElementsByTagName('p');

    3. 对于表单元素，可以通过name查询

      decument.getElementsByName('name');

    4. 通过类名查找

      decument.getElementsByClassName("ClassName");

    5. 通过类找它的子节点

      decument.getElementById('Id').childNodes();
decument.getElementById('Id').children();
